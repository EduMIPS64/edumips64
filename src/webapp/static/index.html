Help

For help on how to use EduMIPS64, please go to Read the docs.

It contains information on the desktop version, but everything except the UI part is applicable to this simulator (with some exceptions like SYSCALL)

Once you load a program, hover over any instruction to see information about it. You will be able to see the address, the instruction's binary representation, the opcode and the CPU stage in which the instruction is in the current step, if it is in the pipeline.

CPU stages are also encoded by colors. (a legend will be added.)

About

This is the web version of the EduMIPS64 CPU simulator.

This is currently work-in-progress, very early stages and not fully functional. See known issues

The core of the simulator is cross-compiled from Java to JavaScript, and the UI is developed with React.

If you are interested in the evolution of this web application or want to contribute to it, please get in touch via GitHub!
