网页用户界面
==================
EduMIPS64 还提供一个完全在浏览器中运行的网页应用程序。模拟器内核由 Java 交叉编译为
JavaScript，并以 Web Worker 的形式运行，而用户界面则使用 React 构建。生产环境部署在
https://web.edumips.org。

本章介绍网页前端。关于源文件格式、指令集、浮点单元和示例程序，请参阅本手册的其他章节：
那些内容与具体的用户界面无关。

快速入门
--------
在代码编辑器中加载或输入程序后，可以使用工具栏单步执行模拟、运行整个程序或重置模拟。
将鼠标悬停在任何指令上可以查看相关信息：地址、二进制表示、操作码，以及（如果该指令
当前位于流水线中）该指令在当前周期所处的 CPU 阶段。CPU 阶段还通过颜色进行编码。

布局
----
网页前端由若干面板组成，分别显示模拟运行的不同方面：

* **代码（Code）** — 加载在内存中的程序，每条指令的地址和十六进制表示。
* **寄存器（Registers）** — 整数寄存器和浮点寄存器的内容。
* **内存（Memory）** — 内存单元的内容。
* **流水线（Pipeline）** — 当前位于 CPU 流水线各阶段的指令。各阶段以颜色编码。
* **统计（Statistics）** — 程序执行相关的计数器（已执行的周期数、已执行的指令数、
  停顿数等）。

加载和运行程序
--------------
程序可以直接在代码编辑器中输入，也可以从内置示例中加载。模拟器支持的源文件格式与
:doc:`source-files-format` 章节中描述的相同。

加载程序后，你可以：

* 执行一个 CPU 周期（单步执行）。
* 运行程序直到它通过 ``SYSCALL 0``（或等效指令）或 ``BREAK`` 指令终止。
* 将模拟器重置到初始状态，同时保留已加载的程序。

帮助与语言
----------
帮助对话框（顶部栏中的问号图标）会在应用程序内打开本手册，左侧带有导航抽屉，并提供
语言选择器，可在英语、意大利语和中文之间切换。

帮助对话框中的 "About" 选项卡会显示模拟器版本以及当前运行版本的说明（生产版、
PR 预览版或本地开发版）。

将 EduMIPS64 作为桌面或命令行应用程序运行
-----------------------------------------
网页前端无需安装即可使用，非常方便；但 EduMIPS64 主要以 Java 桌面应用程序的形式
发布，也可以从命令行运行。桌面 JAR 提供了额外的功能（完整的配置对话框、L1 缓存
模拟器、Dinero 前端，以及用于批处理 / headless 执行的 CLI 选项），这些都记录在
`Read the Docs <https://edumips64.readthedocs.io/>`_ 上的完整手册中。

要安装桌面应用程序或从命令行运行 EduMIPS64，请参阅项目的 GitHub 仓库：

* 项目页面：https://www.edumips.org
* 源代码、发行版和安装说明：
  https://github.com/EduMIPS64/edumips64

如果你发现网页前端中的 bug 或希望提出改进建议，欢迎在 GitHub 上提交 issue。
